# Konsensus

Платформа для разрешения споров с ИИ-медиатором.

## О проекте

Konsensus — это веб-платформа, где пользователи могут конструктивно разрешать споры при помощи искусственного интеллекта. ИИ-медиатор анализирует позиции обеих сторон и предлагает справедливые решения.

### Основные возможности (MVP)

- **Создание спора** — описание проблемы и приглашение оппонента
- **Обмен аргументами** — структурированная подача позиций в раундовой системе
- **ИИ-медиация** — анализ аргументов, определение сильных/слабых сторон
- **Предложение решений** — 2-3 варианта компромиссов от ИИ

## Стек технологий

- **Frontend:** Next.js 15 + React 19 + TypeScript
- **Стили:** Tailwind CSS
- **Backend/БД:** Supabase (PostgreSQL, Auth, Realtime)
- **ИИ:** Claude API (Anthropic)

## Запуск локально

```bash
# Клонировать репозиторий
git clone https://github.com/cwjechw98-lang/konsensus.git
cd konsensus

# Установить зависимости
npm install

# Скопировать и заполнить переменные окружения
cp .env.example .env.local

# Запустить dev-сервер
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000) в браузере.

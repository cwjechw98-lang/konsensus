# Konsensus — Концепция проекта

## Миссия

Создать платформу, где люди могут разрешать споры конструктивно и справедливо с помощью ИИ-медиатора. Konsensus превращает конфликт в структурированный диалог, где каждая сторона может быть услышана.

## Визия

Мир, где споры решаются через аргументы, а не эмоции. Платформа, которая учит людей видеть другую точку зрения и находить компромиссы.

## Ядро MVP (4 основные фичи)

### 1. Создание спора
- Пользователь создаёт спор с описанием проблемы
- Приглашает вторую сторону (по ссылке или email)
- Спор имеет статусы: `open` → `in_progress` → `mediation` → `resolved`/`closed`

### 2. Обмен аргументами
- Каждая сторона пишет свои аргументы
- Структурированный формат: позиция + обоснование + доказательства
- Ограничение по количеству раундов (3-5 раундов)
- Аргументы видны обеим сторонам после отправки обоих

### 3. ИИ-медиатор
- После обмена аргументами — ИИ анализирует позиции обеих сторон
- ИИ определяет:
  - Сильные стороны каждой позиции
  - Слабые стороны и логические ошибки
  - Точки соприкосновения
  - Области принципиального расхождения
- Результат — структурированный анализ (не "кто прав", а объективный разбор)

### 4. Предложение решений
- На основе анализа ИИ предлагает 2-3 варианта решения:
  - Компромиссный вариант
  - Вариант, ближе к позиции стороны А
  - Вариант, ближе к позиции стороны Б
- Стороны могут принять одно из решений или продолжить обсуждение

## Типы аккаунтов

### MVP (базовый, бесплатный)
- Создание споров
- Участие в спорах
- ИИ-медиация (ограниченное количество в месяц)

### Будущее (после MVP)
- **Premium:** неограниченная медиация, расширенная аналитика
- **Professional:** для юристов и медиаторов, кастомные шаблоны
- **Enterprise:** командные споры, интеграции

## Будущие фичи (после MVP)

### Система рейтингов
- Рейтинг качества аргументов
- Рейтинг "конструктивности" пользователя
- Gamification: бейджи за успешные разрешения

### Обучение ИИ
- Сбор анонимизированных данных об успешных разрешениях
- Улучшение качества медиации на основе обратной связи
- Специализация ИИ по типам споров

### Web3 (исследование)
- Запись решений в блокчейн (опционально)
- Токены репутации
- Decentralized dispute resolution

### Другие
- Публичные споры (дебаты)
- Шаблоны для типовых споров (аренда, соседи, бизнес)
- Мультиязычная медиация
- Интеграция с мессенджерами (Telegram бот)

## Монетизация (для MVP: не актуально)

Фокус MVP — валидация идеи. Монетизация планируется через:
1. Freemium-модель (лимит бесплатных медиаций)
2. Premium-подписки
3. B2B-решения для компаний
4. API для сторонних платформ

## Принципы

1. **Нейтральность** — ИИ не принимает сторону
2. **Прозрачность** — логика анализа объяснена
3. **Конфиденциальность** — данные споров приватны
4. **Конструктивность** — платформа поощряет диалог, не конфликт

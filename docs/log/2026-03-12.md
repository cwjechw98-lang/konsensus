## Решения
- Выбран стек: Next.js + Supabase + Claude API
- Причина: популярность, free tier, real-time, серверные компоненты
- Документация на русском, код на английском
- MCP: GitHub + Supabase (оба проверенные, с OAuth)
- App Router + src/ директория для лучшей организации
- Миграция применена напрямую через pg-клиент (Supabase CLI без токена)
- Supabase регион: ap-northeast-1

- Пароль БД содержит спецсимволы — нужно URL-encode (#→%23)
